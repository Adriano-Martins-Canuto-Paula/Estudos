# Repositório de Estudos

Este repositório é dedicado aos **meus estudos em programação**, reunindo exercícios práticos, testes de comandos e desafios de diferentes fontes de aprendizado.

O foco principal é a **prática contínua**, utilizando materiais teóricos apenas como base para a resolução dos exercícios.
