#  Codedex — Cursos e Estudos

Este repositório reúne meus estudos e projetos desenvolvidos durante os cursos realizados na **Codedex**, com foco em programação e inteligência artificial.

O objetivo é documentar meu aprendizado, registrar exercícios, anotações e códigos práticos, além de acompanhar minha evolução técnica ao longo do tempo.
